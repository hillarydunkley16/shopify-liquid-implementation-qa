# minus

```oobleck
number | minus: number
```

returns [number](https://shopify.dev/docs/api/liquid/basics#number)

Subtracts a given number from another number.

##### Code

```liquid
{{ 4 | minus: 2 }}
```

##### Output

```html
2
```

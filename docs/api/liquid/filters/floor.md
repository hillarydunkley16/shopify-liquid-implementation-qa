# floor

```oobleck
number | floor
```

returns [number](https://shopify.dev/docs/api/liquid/basics#number)

Rounds a number down to the nearest integer.

##### Code

```liquid
{{ 1.2 | floor }}
```

##### Output

```html
1
```

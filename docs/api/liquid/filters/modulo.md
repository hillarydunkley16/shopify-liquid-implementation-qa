# modulo

```oobleck
number | modulo: number
```

returns [number](https://shopify.dev/docs/api/liquid/basics#number)

Returns the remainder of dividing a number by a given number.

##### Code

```liquid
{{ 12 | modulo: 5 }}
```

##### Output

```html
2
```

# ceil

```oobleck
number | ceil
```

returns [number](https://shopify.dev/docs/api/liquid/basics#number)

Rounds a number up to the nearest integer.

##### Code

```liquid
{{ 1.2 | ceil }}
```

##### Output

```html
2
```

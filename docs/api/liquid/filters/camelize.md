# camelize

```oobleck
string | camelize
```

returns [string](https://shopify.dev/docs/api/liquid/basics#string)

Converts a string to CamelCase.

##### Code

```liquid
{{ 'variable-name' | camelize }}
```

##### Output

```html
VariableName
```
